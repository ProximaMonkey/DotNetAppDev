ReadMe - Code Samples for Chapter 34, Windows Desktop Applications with WPF

The sample code for this chapter contains one solution with several WPF projects.
The project Formula1Demo contains a Database directory with the SQL Server backup file Formula1.bak. To run the application, you need to restore this database using SQL Server Management Studio.
